Constructs an iterator on the edges starting from a vertex.

Associated Types

type Edges: Iterator<Item=(&'a Self::V, &'a Self::Label)>

The type of the iterator on the edges starting from the vertex

Required Methods

fn edges(self) -> Self::Edges

Returns an iterator which yields a tuple (&'a Self::V, &'a Self::Label) for each edge starting from this vertex. The first element of the tuple is a reference to the destination vertex, and the second element of the tuple is a reference to the label of the edge.
