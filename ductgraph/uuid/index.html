An interface to manage construction of unique clonable handles

A graph structure might be cyclic, and has no clear notion of ownership. Nonetheless, some kind of reference is needed to implement the graph. This module provides an interface to create generic name handles to act as references to vertices in a graph structure. This interface is used only when the handles are generated by the graph structure when a new vertex is created. The users can also build a graph structure which doesn't manage the handles internally, and provide the handle at the time of vertex creation.
